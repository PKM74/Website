NANITE OS

A very Small Operating system designed for X86 Computers. Nanite's Github Page
Runs on older computers such as a Pentium (i586) Class Machine.
I however would recomend atleast a Pentium 2 Class System or higher.

Features

  • Support for the GNU GRUB 2 Bootloader and Multiboot V2
  • Custom Theme for GRUB (Not Complete Yet!)
  • Basic Memory Paging
  • Support for Floppy Disk Drives (FDDs)
  • Basic Serial (RS-232) Support
  • Keyboard Input
Note: Kernel is currently under very heavy development. All things are subject to change at any time.
